# 🏠 HomeVision

**HomeVision** is an innovative web-based platform built with the **MERN stack** (MongoDB, Express.js, React.js, Node.js) to transform the home buying experience. It allows users to explore properties, book visits, download brochures, and directly contact sellers — all through an intuitive, user-friendly interface.

---

## 🚀 Features

### 👤 User
- Secure Registration & Login
- View, Book, and Schedule Property Visits
- Download Property Brochures (PDF)
- Password Recovery via Email
- Contact Property Owners Directly
- View Property Locations via Map Integration

### 🛠️ Admin
- Secure Admin Access
- Add / Update / Delete Properties
- View Bookings & Generate Reports
- Manage User Accounts
- Email Communication with Users
- Visualize Property Locations

---

| Layer        | Technology        |

---

## 📌 System Overview

### Existing System
Traditional real estate platforms often lack real-time booking, direct communication, and ease-of-use.

### Problem Definition
Most platforms are either too complex or don’t offer the features needed for a smooth transaction process.

### Objective
To build a secure, modern real estate platform with:
- Robust property management
- User & Admin authentication
- Direct booking and contact features

---

## 🔍 Project Highlights

- Modern React structure using hooks and context
- RESTful APIs with Express.js
- MongoDB for flexible data modeling
- Scoped CSS and clean UI components
- Git-based version control with organized branches

- **End-Users**: Individuals searching for properties.

## ✅ Advantages

- Smooth, intuitive interface
- Fast property discovery and booking
- Secure login and admin control
- Map and brochure integrations

### ⚠️ Limitations

- Relies on third-party email services for communication
- Requires an active internet connection

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/BhavsarAesha/MERN-Project.git
cd MERN-Project

# Install dependencies for both frontend and backend
cd homevision-backend
npm install

cd ../homevision-frontend
npm install

# Set up environment variables (example: .env)
# MONGO_URI, JWT_SECRET, etc.

# Run backend and frontend
npm run dev     # for backend
npm start       # for frontend
```
